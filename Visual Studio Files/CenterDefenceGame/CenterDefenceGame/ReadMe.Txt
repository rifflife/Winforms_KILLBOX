# 초기 튜토리얼

튜토리얼을 시작합니다.

[W A S D] 로 이동합니다.
[왼쪽 클릭] 으로 무기 1을 발사합니다.
[오른쪽 클릭] 으로 무기 2를 발사합니다.

무기는 총알을 소모합니다.

[ESC] 로 일시 정지합니다.
[F1] 로 디버그 도움말을 볼 수 있습니다.

웨이브 마다 적군이 몰려옵니다.
코어를 방어해야합니다.
블럭을 설치해 최대한 방어하세요.

건설 버튼을 눌러 블럭을 건설하세요.

# 건설 튜토리얼

탄약 생성 블럭은 5초당 1씩 탄약을 생산합니다.
또 탄약 생성 블럭은 최대 탄약 소지량을 10 증가시켜줍니다.
코어는 자체적으로 50 탄약을 소지할 수 있습니다.

자원 생성 블럭은 웨이브가 끝날 때 마다 자원을 1씩 생산합니다.
코어는 웨이브를 빠르게 끝낼수록 더 많은 자원을 생산합니다.

장애물 블럭은 적들이 들어오는 것을 막아주거나 공격을 대신 받아줍니다.

벽 블럭은 적들이 들어오지 못하게 막아줍니다.
때문에 벽 블럭을 설치할 때는 적들을 유인할 입구를 만드는 것이 좋습니다.
적은 벽 블럭 때문에 코어에 도달할 수 없다면 무엇이든지 부수려고 합니다.

# 상점 튜토리얼

자원을 이용해서 플레이어를 업그레이드 하세요.

# 웨이브 튜토리얼

웨이브 시작 버튼을 눌러 웨이브를 시작하세요.

적들을 파괴하고 자원을 획득하세요.


# 웨이브 종료 튜토리얼

웨이브가 종료되면 적들은 모두 사라지고 블럭들은 체력을 회복합니다.
30 웨이브까지 도전해보세요.
